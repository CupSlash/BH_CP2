# Simple Grade Book

This is a program that allows teachers to manage student grades and check how they are performing in their class.

When you run the program, you will see a menu of options.

1. Add New Student
    - Allows you to add a new student to your class.
2. Add Grade to Student
    - Allows you to assign a grade to one of your students.
3. View Student Record
    - Allows you to see how a student is performing in your class.
4. View All Students
    - Allows you to see how all of your students are doing rather than an individual.
5. View Class Summary
    - Allows you to see the average grade in your class.
6. View Class Statistics
    - Allows you to see max, min, and mean for your students' grades.
7. Exit
    - Allows you to leave the application.
